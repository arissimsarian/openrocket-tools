# OpenRocket Tools

A collection of web-based tools for rocket stability analysis and design.

## Rocket Stability Margin Target Analyzer

An interactive tool to help determine the recommended stability margin range for your rocket design.

### Features

- **Stability margin recommendations** based on rocket geometry, flight speed, and wind conditions
- **Launch stability diagnostic** to analyze post-rail wobble causes
- **Thrust curve import** from load cell CSV data (Graphtec DAQ format)
- **Visual charts** showing stability zones and Mach-dependent SM requirements

### Usage

Open `index.html` in a web browser, or visit the live version at:
**https://arissimsarian.github.io/openrocket-tools/**

### Input Parameters

- Body diameter and total length
- Maximum flight speed (Mach number)
- Wind conditions
- Rocket weight and thrust data
- Actual stability margin from OpenRocket
